How to use the Justification Tool

The Explanation View

The upper pane contains a description of why Livingstone assigned a value to a particular model attribute.  This summary is either in terms of an outline (a JTree) or a diagram (a Lattice).  Within the outline or diagram, individual assignments look like this:

<time>#<attributeName>=<value>
Assignments are connected to each other by chains of reasoning, i.e., if Livingstone believed X and Y, then it could conclude Z.  We say belief in Z is justified by belief in X and Y.
